Plugin for automated data migration from Movable Type to WordPress. Convert Movable Type to WordPress easily and fast.

== Description ==

You have your website on Movable Type CMS. Then you have found that WordPress can better suit your needs. Now you are wondering about accurate and flawless Movable Type to WordPress migration. Without having much time or required coding skills, consider to utilize the automated Movable Type to WordPress migration plugin.

= HOW THIS PLUGIN HELPS: =
- Run a Free Demo Migration and observe the new look of your future website;
- It takes approximately 15 minutes to migrate your website data;
- Identify the content items you would like to transfer (Data “Mapping” option);
- The price for the full website data migration depends on the amount of data you are going to move to your new WordPress website.

= WHAT WILL BE MOVED FROM MOVABLE TYPE TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.

= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing Movable Type website to the new WordPress website.

= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
-  For more customized migration type, CMS2CMS offers data migration service packages to order.


== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated MovableType To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
